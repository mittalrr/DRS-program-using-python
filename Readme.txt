A program for DRS review system in cricket using Python and Tkinter. 
DRS review system is a very popular technology in cricket with a good amount of Python logic to implement which makes this program a good source of logic building in Python. 
This technology is used in cricket world cup, IPL and international test cricket. 
This Third umpire toolkit in python gets easy to implement with this awesome Tkinter package.

NOTE:- Add a "clip.mp4" named file containing the video clip of the run out that you wish to watch & review so as to come up with a decision.
